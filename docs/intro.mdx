# POS Payment Integration

##  Quick Start
- Install Android Studio  
- Sync Gradle  
- Setup Payment SDK  
- Start emulator  

##  Projektöversikt
Detta projekt handlar om:
- POS-integration med Payment SDK  
- Betalningar via terminal (off-device) och app (on-device)

##  On-device vs Off-device

###  On-device
- Betalning direkt i appen  
- Snabb och UI-baserad  

###  Off-device
- Extern terminal  
- Stabil i verkliga miljöer  

##  Features
- Flera betalningsflöden  
- Split payments  

##  Miljö
- Kotlin  
- Android  
- Gradle  
- Lokal databas
